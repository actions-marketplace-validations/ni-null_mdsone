# 多國語言

多國語言功能只會在 `-i, --i18n-mode` 啟用時生效，輸出為單一 HTML，內含語系切換。

## 快速使用

```bash
npx mdsone ./docs --i18n-mode=zh-TW -o dist/index.html
npx mdsone ./docs -i=zh-TW -o dist/index.html
```

## 相關參數

| 參數 | 說明 | 範例 |
| --- | --- | --- |
| `-i, --i18n-mode` | 啟用多國語言模式 | `-i` |
| `-i=CODE, --i18n-mode=CODE` | 指定預設語系 | `-i=zh-TW` |

## 環境變數

```env
I18N_MODE=true
I18N_DEFAULT_LOCALE=zh-TW
```

## config.toml

```toml
[i18n]
mode = true
i18n_default_locale = "zh-TW"
```

## 多語內容資料夾結構

`-i` 模式下，輸入資料夾需要是「語系資料夾 -> Markdown 檔案」：

```text
docs/
  en/
    intro.md
    guide.md
  zh-TW/
    intro.md
    guide.md
```

輸出後會以同名文件對齊不同語系內容，並在頁面提供切換。
