# 範本主題

## 內建範本

目前內建範本：`normal`

```bash
npx mdsone ./docs -m --template normal
```

## 變體用法

`--template` 支援 `name@variant`：

```bash
npx mdsone ./docs -m --template normal@default
npx mdsone ./docs -m --template normal@warm-cream
```

## 自訂範本路徑

可直接傳入範本資料夾路徑（也可附加 `@variant`）：

```bash
npx mdsone ./docs -m --template C:\themes\my-doc-theme@ocean
```

## config.toml 對應

```toml
[build]
template = "normal@warm-cream"
```

- `template` 可用 `name`、`name@variant`、或範本資料夾路徑。
- 若同時設定 CLI 與 `config.toml`，以 CLI `--template` 為準。

## 常見範例

```bash
npx mdsone ./docs -m --template normal@default -o docs.html
npx mdsone ./docs -m --template normal@warm-cream -o docs.html
```
